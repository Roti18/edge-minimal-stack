Introduction

EdgeStack is a minimalist, high-performance backend boilerplate for the modern web. Built with Hono, it prioritizes absolute speed.

Vercel Runtime Separation

"Complex logic stays on Node.js, hot data stays on the Edge."

Initial Setup

Ready to deploy in less than 2 minutes.

# 1. Clone
git clone https://github.com/Roti18/edge-minimal-stack.git

# 2. Install
npm install

# 3. Environment
cp .env.example .env

# 4. Develop
npm run dev

Authentication

Google OAuth 2.0 with stateless signatures and HMAC-signed sessions. 

// Auth Endpoints
GET  /auth/google
GET  /auth/google/callback
GET  /auth/session
POST /auth/logout
POST /auth/login

Edge Data API

Globally replicated Edge isolates for sub-10ms response times.

GET /data/config
GET /data/flags
GET /media/:id

/* Performance Header */
Cache-Control: s-maxage=300

Database (Turso)

Distributed SQLite for minimal data latency.

npm run db:init    // Local SQLite
npm run db:migrate // Sync Schema
